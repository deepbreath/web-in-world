HTML5 是新一代 Web 富客户端应用程序开发整体解决方案，包括 HTML5 CSS3 JavaScript API 在内的一套技术组合

应用场景

HTML5 改进 增加大量有意义的语义标签，有利于搜索引擎理解 HTML 内容，HTML5 更结构化，标签更语义话

---- 传统布局 ----

            
            <!-- 头部 -->
            <div class="header">
            <ul class="nav"></ul>
            </div>
            
            <!-- 主体部分 -->
            <div class="main">
                <!-- 文章 -->
                <div class="article"></div>
                <!-- 侧边栏 -->
                <div class="aside"></div>
            </div>
            
            <!-- 底部 -->
            <div class="footer">
            
            </div>

---- HTML5 经典网页布局 ----

            
                <!-- 头部 -->
                <header>
                    <ul class="nav"></ul>
                </header>
                
                <!-- 主体部分 -->
                <div class="main">
                    <!-- 文章 -->
                    <article></article>
                    <!-- 侧边栏 -->
                    <aside></aside>
                </div>

                </footer>

---- 语义标签 ----

            section: 表示区块
            article: 表示文章，评论，帖子，博客
            header: 表示页眉
            footer: 表示页脚
            nav: 表示导航
            aside: 表示侧边栏。如文章的侧栏
            figure: 表示媒介内容分组
            mark: 表示标记（用的少）？
            progress: 表示进度（用的少）？
            time: 表示日期
            与 div, span 没有区别，只是语义上更有代表性，且单标签不用写关闭符号

---- 新语义标签的兼容性处理 ----

引入 html5shiv.js 文件

                
                <!--  条件注释 只有ie能够识别-->

                <!--[if lte ie 8]>
                    <script src="html5shiv.min.js"></script>
                <![endif]-->
            
        
            条件注释解释
            l: less 
            t: than
            e:  equal
            g: great
            ps: 可以使用自定义标签

---- H5 中的多媒体 ----

        autoplay: 自动播放 可以单独设施 autoplay 属性即可 也可以设置成为 autoplay:""
        controls:控制条
        loop: 循环播放
        preload: 预加载，同时设置 autoplay 时候，此属性将会失效
        注意页面自动播放存在安全策略现实，目前测试下来只能用户产生交互才可以进行自动播放

---- H5 中的 DOM 操作 ----

        获取元素相关操作：
        document.querySelector("selector"); 通过 CSS 选择器获取符合条件的第一个元素
        document.queryStelectorAll("selector"); 通过 CSS 选择器获取符合条件的所有元素，
        以数组形式存在

        针类名操作
        Node.classList("class") 添加 class
        Node.classList("class") 移除 class
        Node.classList.toggle("class") 切换 class，有则移除，无则添加
        Node.classList.contains("class") 是否包含当前 class

        自定义属性
        js 中可以使用 class名称 . 属性 类似 classDemo.name 来获取自定义属性
