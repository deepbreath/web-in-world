        存在相同权重，按照顺序覆盖

什么颜色？

层叠性，权重计算问题 
                是否被选中 -> 选中 -> 比较权重 ->
                                    权重不同 -> 展示权重大的
                                    权重相同 -> 展示最后一个顺序的样式
                没有选中 -> 就近原则谁描述的近用谁的 -—> 位置一样 -> 权重比较
CSS 样式的冲突的总结 
                1. 对于相同的选择器。 样式表排序-> 行级别样式 -> 内嵌样式 —>外部样式表
                2. 对于相同类型的样式表 id 大于 类选择 大于 标签选择
                3. 外部样式表 的ID 选择器 > 内嵌样式表的标签选择器
                继承的权重为 0
                总结：就近原则。ID 选择器最大
